Detailed Description

ADC class made for controlling the ADC ports in the MSP432 P401R LaunchPad Development Kit.

Constructor & Destructor Documentation

ADC::ADC ( )

Standard constructor without paramenters. You'll have to add the ADC port you want to use later on using ADC::addPort(DIO_PORT_Even_Interruptable_Type *, uint16_t) or addPort(vector<DIO_PORT_Even_Interruptable_Type*>, vector<uint16_t>)

After setting up the ports you have to use ADC::setup() to set the configuration parameters.
